Automatização de Tarefas

Este projeto automatiza tarefas na Plataforma de Tracking da empresa, utilizando Python para criar um script que realiza a finalização de notas fiscais entregues e a adição de observações em notas fiscais não entregues. As principais bibliotecas utilizadas incluem PyAutoGUI e Pandas.

Jogo do Número Secreto

O "Jogo do Número Secreto" é uma aplicação web responsiva onde o jogador deve adivinhar um número gerado aleatoriamente entre 1 e 100. Com feedback dinâmico e contagem de tentativas, o jogo proporciona uma experiência divertida e desafiadora. Desenvolvido com HTML, CSS e JavaScript.

Calculadora de IMC

A "Calculadora de IMC" é uma aplicação web que permite calcular o Índice de Massa Corporal (IMC) com base no peso e na altura fornecidos pelo usuário. A aplicação é desenvolvida com HTML, CSS e JavaScript, apresentando uma interface intuitiva e orientações sobre a classificação do IMC, como peso ideal, sobrepeso ou obesidade.
